---

1.  Login ke cPanel Anda
    
2.  Buka menu File Manager
    
3.  Klik folder public\_html
    
4.  Blok semua isi yang ada di folder public\_html dan klik menu delete
    
5.  Klik Delete File(s)
    
6.  Tunggu beberapa saat dan selesai.
